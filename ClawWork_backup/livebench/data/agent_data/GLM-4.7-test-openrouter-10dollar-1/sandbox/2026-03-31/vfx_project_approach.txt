# VFX Teleportation Shot - Project Approach and Documentation

## Project Overview
Creating a VFX teleportation sequence for a science fiction film where an actor disappears from a window in a teleportation machine.

## Reference Files
- Base Clip: TWT_001_02.mp4 (220.02 MB, 4K resolution, 23.98 FPS)
- Overlay Clip: TWT_A001_03.mp4 (308.45 MB) - appears to have file corruption issues

## Technical Requirements

### 1. Base Video Processing
- Match dimensions: 3840x2160 (4K)
- Maintain framerate: 23.98 FPS
- Codec: H.264
- Duration and compression matching original source

### 2. Stabilization & Masking (Overlay Clip)
- Apply motion stabilization to remove camera shake
- Create precise mask around window frame
- Generate alpha channel for transparent background
- Isolate actor within window boundaries

### 3. Shot Selection
- Performance footage: ~6 seconds of actor in window
- Empty window footage: ~3 seconds after actor ducks out (from ~20s mark)
- Stitch together for seamless disappearance effect

### 4. Motion Tracking & Compositing
- Track window movement in base clip
- Apply tracking data to overlay clip
- Match scale, position, and perspective
- Ensure seamless integration between clips

### 5. Color Grading
- Match overlay clip color to base clip
- Adjust white balance, contrast, saturation
- Ensure consistency across the shot

### 6. Visual Effects Enhancements
- Flash of light at disappearance moment
- Smoke/mist VFX overlay
- Particle effects for teleportation

## Workflow Steps

1. **Import & Setup**
   - Load both video clips into editing software
   - Verify codec and frame rate consistency
   - Set up project with matching parameters

2. **Overlay Processing**
   - Stabilize motion using tracking points
   - Create window mask (polygon/roto)
   - Extract actor performance with clean alpha

3. **Shot Assembly**
   - Extract 6-second performance sequence
   - Extract 3-second empty window sequence
   - Blend frames at transition point
   - Create smooth fade-out effect

4. **Motion Tracking**
   - Track corners/edges of window in base clip
   - Apply tracking data to stabilized overlay
   - Refine manual adjustments if needed

5. **Compositing**
   - Position masked overlay over window
   - Blend using screen/overlay blend modes
   - Add edge feathering for realism

6. **Color Matching**
   - Analyze base clip color profile
   - Apply LUT or manual color correction
   - Match exposure, contrast, saturation

7. **VFX Elements**
   - Add white flash at disappearance
   - Layer smoke effect (stock footage or generated)
   - Add particle simulation if desired
   - Apply glow/bloom effects

8. **Final Polish**
   - Add subtle camera shake if needed
   - Apply noise/grain to match base
   - Final color pass
   - Export with matching codec settings
